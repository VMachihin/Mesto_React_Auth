# Проект: Mesto

## Описание проекта

Одностраничное веб-приложения для добавления, удаления и выбора интересных фотографий. Так же можно редактировать профиль.

## Используемые технологии:

1. Методология БЭМ.
2. Git.
3. Grid.
4. Flex-box.
5. JavaScript.
6. React

[Для просмотра работы, нажмите](https://mesto.vitmach.ru)
